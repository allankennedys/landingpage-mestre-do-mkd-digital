# Mestre do Digital — Landing Page

Landing page moderna desenvolvida para divulgação da formação **Mestre do Digital**, focada em conversão, apresentação de benefícios, prova social e captação de leads para a turma de 2026.

O projeto foi desenvolvido como peça de portfólio, com foco em design moderno, responsividade, performance e experiência do usuário.

## Preview

A landing page apresenta:

- Hero section com CTA principal
- Sessão de benefícios do treinamento
- Estrutura de módulos e conteúdos
- Depoimentos de alunos
- Sessão de investimento/oferta
- FAQ interativo
- Layout totalmente responsivo
- Interface moderna voltada para marketing digital


## Tecnologias utilizadas

- React
- Vite
- TypeScript
- Tailwind CSS
- Radix UI
- Lucide React

O layout inicial foi prototipado com auxílio do Lovable e posteriormente adaptado e refinado manualmente.


## Objetivo do projeto

Este projeto foi desenvolvido para demonstrar habilidades em:

- Desenvolvimento Front-End moderno
- Criação de landing pages de alta conversão
- Componentização com React
- Estilização responsiva
- Organização de interfaces escaláveis
- Estruturação visual voltada para marketing


## Funcionalidades

- Estrutura SPA performática
- FAQ expansível
- Componentes reutilizáveis
- Navegação fluida
- Sessões otimizadas para conversão
- Design responsivo para mobile e desktop
